## Glosario de Términos

| Término | Definición |
|---------|------------|
| **Bounded Context** | Límite lógico de un dominio |
| **Entity** | Objeto con identidad única |
| **Value Object** | Objeto inmutable sin identidad |
| **Aggregate** | Conjunto de entidades relacionadas |
| **Repository** | Abstracción de persistencia |
| **Smart Component** | Componente con lógica de negocio |
| **Dumb Component** | Componente de presentación |
| **OnPush** | Estrategia de change detection |
| **Signal** | Valor reactivo en Angular |
| **inject()** | Función de inyección de dependencias |
